## Tech Stack

**Client:** React, Redux, TailwindCSS

**Server:** Node, Express

## Run Locally

Clone the project

```bash
  git clone https://github.com/MisbahKhan0009/The-Muse.git
```

Go to the project directory

```bash
  cd The-Muse
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
